# File-Organizer

•	This projects aims to provide the user to login and upload his files. Our system should be able to categorize the files based on the content within the documents.
•	User can create his own account or can login through Twitter and can upload his files. UI is designed using HTML, CSS, BootStrap and JavaScript. Used PHP with WAMP sever with MySQL as database.
•	Used Natural Language Processing and is able to get different entities within the file. Organized the files based on the key entities found and performed sentimental analysis, relevance details upon the entities.
•	Used TwitterOATH API which supports Twitter login and IBM Alchemy API for entity classification.
